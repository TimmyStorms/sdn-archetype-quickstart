h1. Spring Data Neo4j maven quickstart archetype

Generate your Spring Data Neo4j project with this maven archetype. It contains the most used dependencies and some basic unit testing setup.

Use it like e.g. 
@mvn archetype:generate@
@-DarchetypeGroupId=com.github.timmystorms@
@-DarchetypeArtifactId=sdn-archetype-quickstart@
@-DarchetypeVersion=0.1 (or 0.2-SNAPSHOT)@.

Please make sure to add sdn-archetype-quickstart to your local or company maven repository.
